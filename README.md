## Init

```lua
require("lvim-colorscheme").setup()
```

## Default config

```lua
{
    style = "dark", -- dark, darksoft, light
    styles = {
        comments = { italic = true, bold = true },
        keywords = { italic = true, bold = true },
        functions = { italic = true, bold = true },
        variables = {},
    },
    sidebars = {},
    colors = {
        dark = {},
        darksoft = {},
        light = {},
    },
}
```
